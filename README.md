# Show Business – News Portal Website

This project is a static implementation of a news portal homepage dedicated to show business. It is built using semantic HTML5 and modern CSS3.

[Visit the website](https://ahahaharu.github.io/show-business-website/)

## How to Run
1. Clone the repository:
   
```
git clone https://github.com/ahahaharu/show-business-website.git
cd show-business-website
```

2. Install the HTTP Server:

```
npm install --global http-server
```

3. Run the Project:

```
http-server
```

4. Open in Browser:
    
Open your browser and navigate to: http://localhost:8080
